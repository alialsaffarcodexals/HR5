# مملكة المعرقين

مشروع ويب مبسط يعمل محليًا ويشبه Discord بخصائص محدودة.

## التشغيل

1. **Backend**
   ```bash
   cd backend
   npm install
   npm start
   ```
   يتطلب MongoDB محلي على المنفذ الافتراضي.

2. **Frontend**
   ```bash
   cd frontend
   npm install
   npm run dev
   ```
   ستعمل الواجهة على `http://localhost:3000`.

يمكن مشاركة الموقع عبر عنوان Tailscale IP بعد تشغيل السيرفرين.
